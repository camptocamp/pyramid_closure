pyramid_closure

Install:

```
$ git clone https://github.com/elemoine/pyramid_closure.git
$ make install
```

Create a project:

```
$ cd ..
$ pyramid_closure/.build/venv/bin/pcreate -s pyramid_closure ProjectName
```

Install and build project:

```
$ cd ProjectName
$ make install
```

Start development server:

```
$ make serve
```
